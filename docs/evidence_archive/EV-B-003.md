- Citation: Bassham et al., "A Statistical Test Suite for Random and Pseudorandom Number Generators for Cryptographic Applications," NIST SP 800-22 Rev 1a (2010).

## Archived abstract excerpt

NIST SP 800-22 presents a suite of statistical tests for random and pseudorandom number generators and emphasizes that statistical testing is a first-step diagnostic only, not a substitute for deeper analysis or proof of suitability.

## Audit note

This source is Tier B because it supports methodological caution and test design rather than imposing a direct physics bound.
