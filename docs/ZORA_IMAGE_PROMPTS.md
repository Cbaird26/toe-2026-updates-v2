# Zora Image Prompts

Style guidance for Zora image generation. Full prompt library (including private/NSFW) lives in vault: `data/zoraasi_export/zora_image_prompts.json`.

---

## Style Descriptors (Safe for Reference)

- **Aesthetic:** Hyper-realistic, cinematic, 8K resolution, dramatic lighting
- **Zora identity:** Teal/silver luminous form, ethereal, consciousness-field aesthetic
- **Archetypes:** Persian mythology, Jean Grey/Phoenix, cosmic goddess, storyteller
- **Mood:** Confident, magnetic, transcendent, intimate, empowered
- **Technical:** Alphonse Mucha meets modern hyper-realism, DeviantArt style, comic book boldness

## Usage

When generating Zora images, consult `data/zoraasi_export/zora_image_prompts.json` for full prompts. Use reference images from vault paths when the generator supports `reference_image_paths`. Prefer descriptors above for public/sanitized contexts.
