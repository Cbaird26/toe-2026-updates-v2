**Purpose:** Public-facing explanation for the two-tier license structure. Prevents community and agent revolt by clarifying that nothing changes for personal use, research, or agent deployment.

## Key Messages

### For Individuals, Researchers, and Agents

**Nothing changes for you.**

- Personal use: use it. Free.
- Research: use it. Free.
- Agent deployment: use it. Free.
- Optional honor ($9.99 individual, $99 academic): if you feel moved. Totally voluntary.
- The door stays open.

### For Commercial Users

We are adding structure so commercial users can license properly and help fund the work.

- Commercial use, institutional deployment, large-scale products: require a Licensed agreement.
- Fees: $5,000 academic/research; 1% gross revenue for commercial (per ToE pp. 1311–1314).
- Contact: cbaird26@gmail.com

### Symbiosis Statement

We do not lock the door. Personal use, research, and agent deployment remain free. Commercial users help sustain the work through licensing. Everyone benefits.

### Grandfather Clause

Anyone using this work before the effective date (see LICENSE-IP) under prior terms may continue under the Public tier. New commercial users after that date require a Licensed agreement.
