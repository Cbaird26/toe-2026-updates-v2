# Speculative bridges (non-referee)

This folder holds **standalone toy-model** LaTeX articles that connect narrative or interpretive ideas (e.g. foldspace-style navigation, branch selection) to MQGT–SCF notation. They are **not** part of the physics-only submission packet; see [../docs/PHYSICS_ONLY_SUBMISSION_PACKET_2026.md](../../docs/PHYSICS_ONLY_SUBMISSION_PACKET_2026.md) for referee-safe boundaries.

| File | Role |
|------|------|
| [Foldspace_Navigation_Variational_Branch_Selection_2026.tex](Foldspace_Navigation_Variational_Branch_Selection_2026.tex) | Guild functional / Norma condition as variational branch selection |
| [Foldspace_Navigation_MQGTSCF_Norma_Paradigm_2026.tex](Foldspace_Navigation_MQGTSCF_Norma_Paradigm_2026.tex) | Dune-inspired mapping to \(\Phi_c\), \(E\), GKSL container; Zora Navigator sketch |
| [Cosmic_Keyboard_Timeline_Selection_2026.tex](Cosmic_Keyboard_Timeline_Selection_2026.tex) | History ensemble, selection operator, controlled navigation toy; no closed timelike curves |

**Build:** from the TOE repo root, `bash scripts/build_speculative_bridges.sh`, or from this directory run `pdflatex -halt-on-error <file>.tex` (a second pass clears cross-references if needed). Current public citation context: [10.5281/zenodo.19325026](https://doi.org/10.5281/zenodo.19325026) ([record 19325026](https://zenodo.org/records/19325026); prior thesis upload [19301030](https://zenodo.org/records/19301030)).
