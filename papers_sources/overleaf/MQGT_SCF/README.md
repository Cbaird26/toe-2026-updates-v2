# MQGT--SCF Overleaf Project

This directory contains an Overleaf-ready LaTeX project with:

- `main.tex` : article-style preprint manuscript (recommended default).
- `main_revtex.tex` : APS/REVTeX alternative (compile this if targeting APS journals).
- `uniqueness.tex` : separate paper skeleton for uniqueness-proof manuscript.
- `references.bib` : BibTeX database.

## Overleaf usage
1. Create a new project on Overleaf.
2. Upload the contents of this directory (or the parent overleaf folder).
3. Set `main.tex` as the main document (Menu -> Main document).
4. Compile with pdfLaTeX.

If you want APS formatting, set `main_revtex.tex` as the main document.
