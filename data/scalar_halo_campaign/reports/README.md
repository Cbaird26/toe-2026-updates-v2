# Scalar Halo Artifact Bundle

This folder bundles the scalar-halo artifacts generated in this chat.

## Important
Most files here are preserved copies of artifacts that were still present on disk.
A subset of later SWELLS / axis-ratio artifacts had disappeared and were **reconstructed from the metrics explicitly reported in the chat**. Those reconstructed files are marked in `artifact_manifest.csv`.

## Main download recommendation
If you just want everything in one file, use the zip archive:
- `scalar_halo_artifact_bundle.zip`

## Notes
- Reconstructed files preserve the reported summary numbers and named outliers.
- Reconstructed plots are summary visualizations, not guaranteed pixel-identical reproductions of the lost originals.
