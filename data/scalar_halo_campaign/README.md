# Scalar Halo / Cosmic Piano campaign (TOE)

Canonical working tree under the TOE repo for the Scalar Halo research program (weak-field fits, BTFR locking, lensing sanity, SWELLS / J2141 pilots, Cosmic Piano benchmarks) and related figures.

## Layout

| Path | Role |
|------|------|
| `figures/scalar_halo/` | Scalar-halo rotation, lensing, cluster, Jeans, SWELLS, flattened-halo, etc. |
| `figures/cosmic_piano/` | Cosmic Piano LOO / anchored / subset diagnostics |
| `figures/sparc/` | SPARC comparison galleries and residuals |
| `figures/inner_halo/` | Inner-halo correction plots |
| `figures/piano_v2/` | Piano v2 gallery / comparisons |
| `figures/misc/` | `Untitled*.png` and other assets whose stem is only classified as misc |
| `tables/` | `.csv` / `.json` numeric outputs |
| `scripts/` | `.py` analysis drivers and `.tex` notes |
| `reports/` | Narrative `.md` reports |
| `artifact_manifest.csv` | Machine-readable index (regenerated by the organizer) |

**Note:** Cursor chat asset paths are ephemeral UUID-suffixed copies. This directory is the stable home after ingest.

## Refresh from Cursor + Downloads bundle

From the TOE repo root:

```bash
./scripts/scalar_halo_campaign/refresh_scalar_halo_campaign.sh
```

Or only the Python ingest (does not delete existing figures first):

```bash
python3 scripts/scalar_halo_campaign/organize_assets.py
```

Behavior:

1. **Cursor assets** (default: `~/.cursor/projects/Users-christophermichaelbaird-Downloads-mqgt-frequency-atlas-bundle-2/assets`): copies only PNGs whose logical basename starts with `scalar_halo_`, `cosmic_piano_`, `sparc_`, `inner_halo_`, `piano_v2_`, or `untitled` (case-insensitive), strips trailing `-UUID` before `.png`, and routes by theme.  
2. **Downloads bundle** (default: `~/Downloads/scalar_halo_artifact_bundle`): gap-fills `tables/`, `scripts/`, `reports/`, and any campaign PNGs not already present. Skips the bundle’s legacy `artifact_manifest.csv`.

Environment overrides:

- `SCALAR_HALO_ASSETS` — Cursor assets directory  
- `SCALAR_HALO_BUNDLE` — bundle directory for gap-fill  

## Ingest from `~/Downloads/UPDATE`

March 2026 work may land first in an all-caps **`UPDATE`** folder under Downloads. To inventory everything there, copy campaign PNGs here (and finance screenshots to `docs/internal/finance_os/screenshots/`), and merge new rows into `artifact_manifest.csv`:

```bash
python3 scripts/update_folder_ingest.py
```

See `data/update_catalog/README.md` for manifest output paths and `UPDATE_DIR` override.
